# Core Demo

Basic integration showcase for the JavaScript `kcli` SDK plus the alpha demo
SDK module.

This demo shows:

- executable-local top-level options such as `--verbose` and `--output`
- imported inline parsing through `alpha.getInlineParser()`
- alias rewrites such as `-a -> --alpha-enable`
- executable-local usage/composition logic inside `main.js`
- built wrapper execution through `./demo/exe/core/build/latest/test`
